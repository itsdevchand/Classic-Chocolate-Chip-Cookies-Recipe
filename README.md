# Classic Chocolate Chip Cookies Recipe Page

A clean, semantic, and modern web page showcasing a classic chocolate chip cookie recipe. This project demonstrates best practices in HTML hierarchy and CSS styling for content-heavy text layouts, ensuring great readability and responsiveness.

## 🚀 Features

- **Clear Information Hierarchy:** Uses structured headings (`<h1>`, `<h2>`) to guide the reader's eye naturally.
- **Semantic Lists:** Utilizes unordered lists (`<ul>`) for ingredients and ordered lists (`<ol>`) for clear, step-by-step instructions.
- **Responsive Design:** Optimized for both desktop and mobile viewing with CSS Media Queries.
- **Modern Typography & Palette:** Uses a warm, food-inspired color palette (creams, warm oranges, and charcoal text) for long-form readability.
